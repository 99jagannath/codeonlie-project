FACTORIAL OF NUMBER

description of question

In mathematics, factorial of a given number is given by the multiplication of numbers from one upto that number

0!=1,1!=1,2!=2,3!=6,4!=24,5!=120,6!=720......

   #include<stdio.h>
   
int main()
{
  printf("Enter a number for finding factorial");
  int n,fact=1,i;
  scanf("%d",&n);
  for(i=1;i<=n;i++)
  {
    fact*=i;
  }
  printf("Factorial of %d is %d",n,fact);
  return 0;
}
